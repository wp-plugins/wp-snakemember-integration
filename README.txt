Wordpress integration plugin for SnakeMember

== Description ==

This plugin allows you to integrate your Wordpress installation with your current admin account on Snakemember. This will allow you to protect pages, track your affiliate events, use campaign webforms, and much more.

== Installation ==

1. Upload the contents of the zip file (`wp_snakemember folder`) to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure the API credentials on the 'Snakemember' menu.
